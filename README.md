### 2. File Organizer App

**Location:** `file-organizer/`

An intelligent file organizer that automatically categorizes files by type.

### Features
- Automatic file categorization by extension
- Support for multiple file types:
  - **Documents:** `.txt`, `.pdf`, `.doc`, `.docx`
  - **Images:** `.png`, `.jpg`, `.svg`
  - **Audio:** `.mp3`, `.wav`
  - **Video:** `.mp4`
  - **Compressed:** `.zip`, `.rar`, `.tar`
- Individual file operations
- File information display
- Safe directory creation
